Why json iterator (jsoniter)?

Json iterator is a novel JSON parsing library with best performance

  • It can bind data to your class/struct, just 10x faster (see benchmark)
  • Unique iterator api allows you work with JSON directly in a streaming way (how streaming works)
  • Mix bind api and iterator api seamlessly (see integration)
Also, it is has both Java and Go version with same api and performance.
