Why json iterator (jsoniter)?

  • Jsoniter is the fastest JSON parser, it could be up to 10x faster than normal parser, data binding included (see benchmark)
  • Having a developer friendly api is our #1 prioprity, you can choose from bind-api, any-api or iterator-api (checkout jsoniter api choices)
  • Unique iterator api can iterate through JSON directly, zero memory allocation! (see how iterator works)
Available today, as both Java and Go version.
