json iterator (jsoniter) 有何独特之处？

Json iterator 总是给你最好的性能

  • 把 json 数据绑定到 class/struct 上，但是比普通的库快10倍 (这里是 性能评测)
  • 独特的 iterator api 让你可以用流的方式直接读取JSON (来看看 流式解析 的魅力吧)
  • bind api 和 iterator api 无缝集成 (灵活的 集成方式)
同时提供 Java 和 Go 两个版本，api 相同而且性能都很棒哦。
