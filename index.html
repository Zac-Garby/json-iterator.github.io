Why json iterator (jsoniter)? !!!!!

Json iterator is a novel JSON parsing library with best performance

  • Bind JSON data to your class/struct, but 10x faster (see benchmark)
  • Unique iterator api to work with JSON directly in a streaming way (how streaming works)
  • Mix bind api and iterator api seamlessly (see integration)
Both Java and Go version available, with exactly same api and equally best performance.
