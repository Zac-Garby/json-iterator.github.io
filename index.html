Why json iterator (jsoniter)?

  • Jsoniter is fastest JSON parser, it could be up to 10x faster than normal parser (see benchmark)
  • Having a developer friendly api is our #1 prioprity (checkout jsoniter api)
  • Iterator api iterate through JSON directly, zero memory allocation! (see our unique iterator api)
Available today, as both Java and Go version.
